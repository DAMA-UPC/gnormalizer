### Developing

#### Run the test

For running all the project Unit tests:

```sh
sbt test
```

For running the failed tests from the latest build:
 
```sh
sbt testQuick
```

#### Check for library updates: 

For checking if there is any dependency update: 

```sh
sbt dependencyUpdates
```
